About GyanGuru

GyanGuru is an AI-powered Machine Learning learning assistant designed to help students, developers, and enthusiasts master ML concepts through interactive text explanations, code examples, audio lessons, and visual diagrams.

Our Mission

Our mission is to make Machine Learning accessible and understandable to everyone. We leverage the power of AI technologies like Google Gemini and Stable Diffusion to create personalized, interactive, and visually rich learning content.

Features

  • AI-powered text explanations for all ML topics
  • Working Python code generation with examples
  • Audio lessons for learning on the go
  • AI-generated visual diagrams for complex concepts
  • Secure and private API key management
  • Open-source and community-driven

Technologies Used

GyanGuru is built using:

  • Flask (Python Web Framework)
  • Google Gemini API for AI content generation
  • Stable Diffusion for image visualization
  • HTML, CSS, and JavaScript for frontend
  • Open-source libraries for ML and AI integrations

We welcome feedback and contributions. You can reach us via our social links in the footer or contribute directly on our GitHub repository.
